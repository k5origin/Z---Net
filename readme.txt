This program uses the UDP Protocol to transfer files between two computers via a router. There are settings to simulate random delays and packet losses as well as countermeasures to compensate for their occurances. 

Start the programs in the following order: Server, Router, Client
On the client file, enter the command 'get' followed by the name of the file (including the extension) in the 'server' folder you wish to transfer to the 'client' folder and press ENTER
